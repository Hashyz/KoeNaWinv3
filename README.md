# KoeNaWin - Myanmar Buddhist Prayer Beads

A beautiful 3D web-based prayer beads (rosary) application for Myanmar Buddhist meditation traditions.

## Features

### 3D Prayer Beads
- **108 Traditional Beads** - Authentic mala configuration with guru bead and tassels
- **9 Child-Friendly Beads** - Larger, easier-to-count mode for children
- Interactive sequential counting with smooth animations
- Beautiful sandalwood-inspired visual design

### 9x9 Magic Square Grid System
- Traditional 83-day meditation cycle (81 practice days + 2 rest days)
- Daily round tracking based on Buddha's 9 attributes
- Automatic cycle goal calculation (1-9 cycles per day)
- Vegetarian day alerts (every 9th column)
- Rest day detection (days 82-83)

### User Experience
- Mobile-responsive design with touch controls
- Haptic vibration feedback on bead selection
- Celebration overlay when daily goal is achieved
- Myanmar timezone support (UTC+6:30)
- Progress persistence with localStorage
- Automatic midnight rollover detection

## Technology

Built with:
- Three.js for 3D graphics
- Vite for development and building
- Pure JavaScript (no frameworks)
- CSS3 animations

## Getting Started

```bash
npm install
npm run dev
```

## Usage

1. **Set Start Date** - Click the settings icon and select a Monday to begin your 83-day cycle
2. **Count Beads** - Tap beads sequentially to count your prayers
3. **Track Progress** - Watch your daily cycle count and overall progress
4. **Switch Modes** - Toggle between 108 beads (traditional) and 9 beads (child mode)
